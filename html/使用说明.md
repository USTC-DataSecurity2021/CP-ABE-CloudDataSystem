# 使用说明



### 环境

+ python == 3.7
  + charm-crypt
+ php > 7 + apache2



> 注：apache使用默认目录/var/www/html，目录需有完全读入读写权限
>
> 访问网址http://ip:port/login.html



### 使用



#### 注册



+ 输入用户名密码学校
+ 复制返回的pk,sk并保存至本地文件中



#### 登陆

+ 输入用户名密码登陆



#### 下载

+ 点击右侧的下载按钮下载文件
+ 用本地的解密脚本解密



#### 上传

+ 通过界面上方的"选择文件"选择要上传的文件。
+ 通过"访问结构"输入指定的访问结构。
+ 点击 Submit上传。
